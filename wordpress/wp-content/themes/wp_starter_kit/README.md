WP Starter Kit
===

Introducing WP Starter Kit.  A WordPress starter theme built atop Google's Web Starter Kit.

* Well coded, documentated and modern HTML5 templates.
* DRY, not WET theme coding with the WordPress theme wrapper
* Strict attention to detail

This theme is a project of the Advanced WP (AWP) group. We are a band of WordPress enthusiast looking to give back to the WP community through code, meetups, WordCamps, and more. Join us at https://www.facebook.com/groups/advancedwp/
